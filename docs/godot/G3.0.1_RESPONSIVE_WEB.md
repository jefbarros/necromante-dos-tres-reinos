# G3.0.1 - Responsividade Web

## Objetivo

Adicionar responsividade minima obrigatoria ao MVP Godot G3.0 para que o build Web seja jogavel e legivel em desktop, notebook e mobile horizontal, com limitacoes claras para mobile vertical.

Esta etapa nao adiciona sistemas novos de gameplay. O foco e layout, escala, HUD, input minimo e documentacao.

## Resolucoes-alvo

- Desktop 16:9: 1920x1080.
- Notebook/tela menor: 1366x768.
- Base do projeto: 1280x720.
- Mobile horizontal: 844x390.
- Mobile vertical: 390x844, experimental e limitado.

## O que foi ajustado

- Mantido viewport base 1280x720.
- Configurado stretch Web com `canvas_items` e `expand`.
- HUD passa a recalcular posicoes conforme o viewport.
- Painel principal, inventario, status, controles e overlay central sao reposicionados para evitar sair da tela.
- START, VICTORY e GAME_OVER aceitam `Enter`, clique ou toque para iniciar/reiniciar.
- Mobile/notebook exibem texto de controles mais curto.
- Mobile compacto exibe botoes touch minimos para ataque, servo e pocao.

## Limitacoes mobile

- Mobile Web ainda e experimental.
- O alvo principal do MVP G3.0 continua sendo desktop/notebook.
- Mobile horizontal e recomendado.
- Mobile vertical pode carregar e exibir HUD basico, mas a experiencia de combate e limitada.
- Controles touch completos, joystick virtual e UI mobile dedicada ficam fora do escopo desta etapa.

## Comandos de validacao

```bash
npm.cmd run check
git diff --check
git diff --name-only -- src
git diff --name-only -- docs\legacy-html5 src
git ls-files | findstr /i "export_presets.cfg"
godot/Godot_v4.6.2-stable_win64.exe --headless --path godot --quit-after 2
godot/Godot_v4.6.2-stable_win64.exe --headless --path godot --export-release "Web" ../docs/play-godot/index.html
```

## Teste no navegador

1. Servir o build Web local:

```bash
python -m http.server 8125 --directory docs/play-godot
```

2. Abrir `http://127.0.0.1:8125/index.html`.
3. Testar em 1920x1080, 1366x768, 1280x720, 844x390 e 390x844.
4. Confirmar que a pagina carrega, START aparece, a run inicia, HUD fica visivel, player move, ataque funciona e inventario abre.
