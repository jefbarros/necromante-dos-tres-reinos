# Concept Art

Esta pasta guarda imagens de referência e concept art usadas apenas para documentação visual de `Necromante dos Três Reinos`.

As imagens aqui não são carregadas pelo jogo em runtime. Elas servem para registrar clima, paleta, leitura visual e nível de detalhe desejado. Arte final, símbolos, personagens, mapas, composições e assets do jogo devem permanecer originais.

Se uma imagem virar asset real de runtime no futuro, ela deve sair desta pasta e ir para `assets/images/`, com licença e origem documentadas.
