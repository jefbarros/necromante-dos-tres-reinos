## Como Testar

1. Abra a URL no navegador
2. Aguarde o carregamento do Godot
3. Use WASD ou setas para mover
4. Use Espaço ou clique para ataque
5. Verifique se inimigo(spawn) aparece
6. Verifique console(F12) para erros
